# Contributing to Awesome Agent Orals

Thank you for your interest in contributing! This list is maintained by the community.

## Inclusion Criteria

A paper must satisfy **both** conditions:

1. **Presentation type:** Accepted as an **Oral**, **Spotlight** (NeurIPS), **Best Paper**, **Outstanding Paper**, or **Best Resource Paper** at a CCF-A AI conference:
   - ICLR, ICML, NeurIPS, ACL (+ EMNLP, NAACL if added), CVPR (+ ICCV, ECCV if added)

2. **Topic:** Clearly related to **AI agents**, including but not limited to:
   - LLM / VLM / multimodal agents
   - Multi-agent systems
   - Embodied agents & robotics
   - Agentic workflow & automation
   - Tool use & function calling
   - Agent safety, alignment, and governance
   - Agent benchmarks and evaluation

## How to Contribute

1. **Fork** this repository and create a new branch.
2. **Add the paper** in the appropriate conference section of `README.md`.
   - Use the existing table format.
   - Use the correct badge: 🏆 Best Paper / 🥈 Runner-up / ⭐ Outstanding / 📖 Oral.
   - Include the official conference link and arXiv link if available.
   - Write a concise one-sentence TL;DR (≤ 25 words).
3. **Add to the Topic Index** under the relevant category.
4. **Submit a Pull Request** with a brief description of what you added and why it qualifies.

## Style Guide

- Paper titles in **bold**.
- Authors: First author + senior/corresponding author. Add institution if notable.
- TL;DR: one sentence, present tense, focus on the key contribution.
- Keep the conference sections in reverse-chronological order (newest first).
- Do not add papers that are only workshop papers, posters, or preprints.

## Reporting Issues

If you find an error (wrong link, incorrect award label, misclassified topic), please open an issue with the paper title and the correction.
